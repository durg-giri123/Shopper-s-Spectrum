# Shopper Spectrum

Customer Segmentation and Product Recommendation System using Machine Learning.

## Features

* Customer Segmentation using RFM Analysis
* K-Means Clustering
* Product Recommendation System
* Interactive Streamlit Dashboard

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit
* Joblib

## Run Project

```bash
streamlit run app.py
```

## Note

The similarity_matrix.pkl file is excluded from GitHub due to GitHub's 100 MB file size limit. The recommendation system works correctly when the file is available locally.
